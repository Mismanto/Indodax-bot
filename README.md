# Indodax Bot

### About
bot discord for check price crypto coins | API Indodax

### Info
###### API :
[API indodax](https://indodax.com/downloads/BITCOINCOID-API-DOCUMENTATION.pdf)

### use
|   Commands    |    Discord    |    Telegram    |
| ------------- | ------------- | ------------- |
|  help  | i>help | /help |
| about  | i>about | /about |
| price | i>price <name_coin>` | /price <name_coin> |

 ```

 ```


 ```
 rake run:discord
 ```
run bot telegram :
 ```
 rake run:telegram
 ```
